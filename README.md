## Prerequisites
### pnpm
[pnpm](https://pnpm.io/installation): Fast, disk space efficient package manager.
```bash
npm install -g pnpm

```
### Nest CLI
Setting up a new project is quite simple with the [Nest CLI](https://docs.nestjs.com/first-steps).

```bash
npm i -g @nestjs/cli
$ nest new

pnpm install
pnpm run start:dev


```
